# 🌐 Portafolio Maestro - Fernando JAZ

Este repositorio contiene el código fuente de mi sitio web personal y centro de mandos profesional, desplegado mediante **GitHub Pages**.

## 🚀 Propósito
El objetivo de este proyecto es centralizar mi ecosistema digital, conectando mis servicios profesionales de optimización técnica con mis experimentos creativos en JavaScript.

## 🏗️ Estructura del Ecosistema
- **Home:** Landing page con diseño adaptativo.
- **Proyectos Profesionales:** Repositorios de servicios técnicos (Fontanería, Reformas).
- **Laboratorio Creativo:** Espacio de experimentación con la API de Canvas y CSS3.

## 🛠️ Tecnologías utilizadas
- **HTML5:** Estructura semántica.
- **CSS3:** Diseño responsivo, Grid, Flexbox y animaciones.
- **JavaScript:** Lógica de navegación y efectos dinámicos.
